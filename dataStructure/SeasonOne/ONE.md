## 算法
- 算法是用于解决特定问题得一系列的执行步骤.
```javascript
// 计算a+b的值
function add(a,b){
    return a + b
}
// 计算1+2+3+4+5+...+n的和
function sum(n){
    let result = 0;
    for(let i = 1;i <= n;i++){
        result += i;
    }
    return result;
}
```
